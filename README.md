## 💻 Sobre o projeto

Api Fipe é uma API em Java, com funcionalides de pesquisar valor de veiculo na tabela fipe via prompt.

---

## ⚙️ Funcionalidades

- [x] Pesquisar preço de veiculo na tabela fipe
